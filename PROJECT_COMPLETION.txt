🔐 AUTHENTICATION & AUTHORIZATION

Login Redirection:
  • Admin user → /
  • Caretaker user → /caretaker
  • Warden user → /warden
  • Student user → /student

Route Guards:
  • AdminRoute → isAdmin only
  • CaretakerRoute → isCaretaker || isAdmin
  • WardenRoute → isWarden || isAdmin
  • StudentRoute → isStudent only

Role Inheritance:
  • Admin can access ALL routes
  • Caretaker can access /caretaker only
  • Warden can access /warden only
  • Student can access /student only
